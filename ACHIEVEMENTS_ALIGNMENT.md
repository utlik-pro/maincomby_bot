# Рассинхрон Достижений (Achievements Discrepancy)

Обнаружено несоответствие идентификаторов (ID) и названий достижений между кодом бота и Mini App. Это приведет к тому, что достижения, выданные ботом, не будут отображаться в приложении (или будут отображаться некорректно).

## Сравнительная таблица

| ID в Python Bot (`services/notifications.py`) | ID в Mini App (`src/types/index.ts`) | Название (Bot) | Название (App) | XP (Bot) | XP (App) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| `first_event` | **`first_step`** | Первое событие | Первый шаг | - | 50 |
| `networker` | `networker` | Нетворкер (5 матчей) | Нетворкер (25 матчей) | - | 200 |
| `regular` | `regular` | Постоянный участник (5) | Завсегдатай (10) | - | 150 |
| `social_butterfly` | `social_butterfly` | Душа компании (10) | Душа компании (10) | - | 150 |
| `veteran` | `veteran` | Ветеран (10 ивентов) | Ветеран (1 год) | - | 500 |
| `helper` | *Отсутствует* | Помощник | - | - | - |
| `profile_complete` | *В коде есть `XP_REWARDS`* | Полный профиль | - | - | 25 |
| `early_bird` | *Отсутствует* | Ранняя пташка | - | - | - |
| *Отсутствует* | **`on_fire`** | - | На волне (3 подряд) | - | 100 |
| *Отсутствует* | **`critic`** | - | Критик (5 фидбеков) | - | 75 |
| *Отсутствует* | **`sniper`** | - | Снайпер (100% посещ.) | - | 200 |

## Рекомендованный план исправлений

Необходимо привести `app/services/notifications.py` в соответствие с типами в Mini App, так как App является "витриной" для пользователя.

1.  **Переименовать ID в боте:**
    *   `first_event` -> `first_step`
2.  **Синхронизировать условия выдачи:**
    *   `networker`: В боте выдается за 5 матчей, в аппе описание "25 матчей". Решить, какое число правильное (скорее всего 5 для начала легче).
    *   `veteran`: В боте "10 мероприятий", в аппе "1 год". Разные механики.
3.  **Добавить недостающие в App:**
    *   Если `helper` и `early_bird` важны, добавить их в `ACHIEVEMENTS` в `miniapp/src/types/index.ts`.
4.  **Реализовать логику выдачи новых ачивок в Боте:**
    *   Написать логику для `on_fire`, `critic`, `sniper`.
